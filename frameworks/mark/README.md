# Mark

Mark is a high performance API micro framework based on FastRoute and workerman helps you quickly write APIs with php. The core code is only about 200 lines.

https://github.com/passwalls/mark
https://github.com/walkor/workerman
https://github.com/nikic/FastRoute

## Stack

- **Language:** PHP
- **Engine:** Workerman (event)

## Notes

- Use nikic/fast-route a fast PHP Regex router
